Investigating the Titanic Disaster¶

In this project I will use a dataset based on the passengers list of the titanic last trip. This data shows personal details of 891 of the 2208 passengers; things like Name, age and sex, but also their sex, their cabin number, whether they embarked or not, or even if they survived. There are more details to all passengers in here, and it is interesting to find a rich dataset like this one of an event that happened not too long ago in the relative human history, but before the age of technology, where technology could have made a big difference. What I hope to find is if there are any important characteristics to best classify whether a passenger would have likely survive or not.

Replacing categorical missing values¶

Now columns like 'Survived' and 'Pclass', although they contain int data types, are still categorical features. They designate each passenger a label pertaining to a set of categories, and not a value on a spectrum, so it would be wise to simply eliminate these missing values at times of data handling. The 'csbf' and 'cdbf' functions will filter out these rows with missing values for rows of interest.

def csbf(feature):
    x = s[pd.notnull(s[feature])]\
        .groupby([feature])['Survived'].count().to_dict()
    print 'survivors by %s =' % (feature),x
    return x


### Counting deceased by feature

def cdbf(feature):
    x = d[pd.notnull(d[feature])]\
        .groupby([feature])['Survived'].count().to_dict()
    
    print 'deceased by %s =' % (feature),x
    return x


### Calculating probability of survival by feature group

def survival_prob_by_feature_group(feature, d1, d2):

    print 'percent of survivors by %s: \n' % (feature)
    
    for i in d1:
        # Here we calculate and print to console the percentage of survivor 
        # per feature group
        print '\t'+str(i)+' class:',\
        str(d1[i]/float(d1[i]+d2[i])*100)+'%'

    
### Graph percentage-categorical data

def graph_pcat_data(a, title, column_name):
    
    a = a.groupby([column_name])['Survived'].count().to_dict()
    b = td.groupby([column_name])['Survived'].count().to_dict()
    
    # c will be a dictionary containing our feature's classes as keys, and their
    # respective percentage values, as values.
    c = {i:(a[i]/float(b[i])) for i in a.keys()}
    
    #matplotlib will help us plot our data
    plt.bar(range(len(c)), c.values())
    plt.xlabel(column_name)
    plt.ylabel('percent (%)')
    plt.xticks(range(len(c)),c.keys())
    plt.title(title)
    plt.show()

### function for new Adult column

def if_adult(age):
    if age >= 18:
        return 'adult'
    elif age < 18:
        return 'minor'

Report¶

In this project I was looking for a characteristic described in the data that best predicted probability of survival per each passenger. I hypothesized that passenger class would be very important in this task since class would best describe the areas where each passenger inhabit the most. For example a first class passenger might have spent most of their time in areas closest to the life raft, atop the deck, even at the time of the accident. From the data on these passengeres we can see that those in 1st class had about a 3x higher rate of survial as those in 3rd class.

Then I looked at the survival rates of passenger separated by sex. It seems as females passengers had a close to 4x times higher rate of survival as males did. This should come as to no surprise; we've all heard the phrase "Women and children first!" somewhere, which led me to believe that another important factor might have also been age.

I decided to create a new column in the original td dataframe. This new feature would describe each passenger as an 'adult' or 'minor'. This way they could be separated and examined. Here I found that both minors had a higher probability of survival, that being ~54%, whereas adults had a ~36% probability.

The difference between the sample and the population, we don't know how the sample was chosen from the actual population of people that were on the Titanic. There could have been some sort of intentional or unintentional bias in how the sample was selected. You need to state what could be your future work or potential areas to explore. Were there some factors limiting your analysis. All this should be clearly stated

However, like all data, this set is in no way perfect. There were 2,208 passengers (and crew) abord the titanic in it's last trip. Given that we only have data for 891 passengers, we should bare in mind that we do not know how this sample data was collected, and if there were any biasses in the process of selection. Furthermore, some future work may include: tracing back relatives on-board for each passenger and analysing whether being part of a family made you more likely to survive.

Some factors limiting my analysis were lack of information; not just lack of passenger in the dataset, but also lack of variable values for those passenger already present in the dataset. Other important factors are lack of variables (features), or information on each passenger; maybe presence of specific medical conditions would serve as better markers in calculating rates of survivals.
